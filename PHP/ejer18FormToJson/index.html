Ayudas para el ejercicio

El div contenedor deberá ser desactivado cuando se activa la ventana modal.
Desactivar el contenedor podría ser darle transparencia y deshabilitar todos sus botones.


Desactivado:

pointer-events: none
opacity: 0.3


Activado:

pointer-events: auto
opacity: 1
